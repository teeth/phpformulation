Now we will start with the first section of the PHP script. This is setting up our variables and
referencing the input field names of the contact us page. Read the description of the code
and type in the code following it. This will be the structure of the tutorial.

The $isSent variable will be the indicator whether the content of the web
form has been sent. It is set to false to signify that no form has been sent with the email address.


The $hasError variable is used to print an error message if there are any errors in
the input fields. It is set to false by default to signify there are currently no errors, when it’s true an
error message will be written above the form in red. The error message will be discussed in later on
in the tutorial.


The $isURLAllowed checks that the URL supplied with the email is permitted. This
check prevents spam from coming through your inbox. You can set it to true if you want to allow URL
to be supplied to the form.


The $fields variable is important part of this section of code. This variable
compiles an array of the input fields names used in the contact us page. The input field name comes
from the “name” name of the input field. So for example the Email Address input field name has an “name”
name called email, so the name email will be referenced in the
list of arrays. So when the input fields have been referenced they will be stored in the
$fields variable. This will be used later in the script for printing out the content
entered in the input fields.
