Lesson 4 - Post the Fields

This part of the PHP script is quite simple. What you’re doing is iterating over the array variable
$fields and storing the data into the variable $message. This will be how your fields of data are
posted.

The foreach loop below is looping over each of the array of fields. These fields are the data the user
has entered in each of the input fields of the contact us page. It stores the fields in the variable
$field.


The variable $message will hold the stored array of $field; this variable will be used in the later part
of the script. ucwords will capitalise the first character of each word, i.e. name becomes Name. The
variable $_POST collects values in the web form. The web form uses the method="POST" attribute
(take a look at the contact us page code for the method). So this will be used as a post method for
the submission of the form.
