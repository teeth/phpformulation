Lesson 1 - Create Web Form

We will start of by creating a simple web Form, which will be used to base our PHP script on.
Readers of this tutorial might already have a website with a web form and are looking to apply the
PHP script to their page, but for the benefit of the readers who haven’t created a web form already,
we will start this tutorial with a design of a simple web form. Also a note, none of the codes in this
tutorial will be available for download; the codes have to be copied line by line. This is beneficial for
readers to learn the process of building the code step by step. If the code was just offered for
download, people probably won’t learn much.

Below is a screen shot a Web Form which will be utilised as a contact us page. It contains seven input
fields to it, which are important elements when we go on to the PHP scripting.



The screenshot shows a web form that contains input fields which includes text fields, a drop down
list and a radio button. Plus two buttons: the reset button to empty the fields and a submit button to
submit the form. The text fields consists of fields where the user can enter their Name, the
Organisation they represent, their Email Address, Telephone Number and a large space where they
can type in their concerning Message. As well as the text fields, there’s the drop down list for users
to select a Subject for their message and a radio button to Rate the Website they visited.

If you noticed the words that are bold, they are the seven input fields which are going to be
referenced in the PHP. These will be the information that will be sent to your email account.
